My Favorite Apps

Hello! This a list of my favorite apps. I use these all quite often and all of them listed here are very useful both for work-related things and for leisure. I will give a brief description of each and discuss their pros and cons as well.

Google Voice

An app that does the main functions a phone is based on. Things like calling, texting and storing voicemail.

Pros Cons
Free for personal use

Can be used on multiple devices

Can choose a number from a variety of area codes

Can be used to make calls from a computer

Great for separating work and personal calls

 Can be a pain to set-up properly

Requires an existing phone number

Get it here

NewPipe

A youtube client/wrapper that removes annoying features in the normal youtube app

Pros Cons
Free

Adds features like downloading and background playing

Gives users more anonymity

 Can't use your youtube account

Can't comment or like videos

Has occasional network errors

Get it here

Pydroid 3

A python interpreter that offers as close to a full PC experience as I have seen on a mobile app

Pros Cons
Can use the terminal like on a PC

Package manager is very easy to use

Only mobile app I found that can run jupyter notebooks well

 Need to get the pro version to use it well

Annoying to set-up environments

Get it here
